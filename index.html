General Attributes

Attribute Name Description
class Class An HTML attribute used to select one or more elements for the application of styling or logic.
dir Direction Sets the direction of the text of an HTML element.
id Unique Id Used to specify a unique id for an HTML element, which cannot be shared across multiple elements within the same HTML document.
src Source An HTML attribute used to specify the location of a digital resource.
style Style Specifies how an element will be rendered in a web browser.
title Title Contains and can display the specified information about the element it belongs to.

Tags

Tag Name Description
<h1,h2,h3...> Heading Among the HTML elements are the 6 heading elements. They are labeled 1-6, with 1 being the largest sized heading and 6 being the smallest.
<ol> Ordered List Represents an ordered list of items.
<ul> Unordered List Represents an unordered list of items.
<menu> Menu Represents an unordered list of items with more semantic meaning than a regular <ul> element.
<li> List Item Represents a single list item. This element must be wrapped in an <ol>, <ul>, or <menu> element.
<q> Short Quotations Represent a brief inline quotation
<blockquote> Blockquote Represent a longer quotation
<b> Bold Represents text that the readerâ€™s attention should be directed to, without further significance. Usually displayed with boldface.
<em> Emphasized Represents text which is emphasized. Usually displayed with italics.
<i> Italic Represents text set off as an idiomatic expression, technical term, taxonomical designation and so on. Usually displayed with italics.
<stong> Strong Represents text that is urgent or important. Usually displayed with boldface.
<u> Underline Represents text to be displayed with a non-textual annotation. Usually displayed using a solid underline.
<!--> Comment Used for documentation and excluding markup from being rendered.
<img> Image Displays an image on the web page.
<img> Image Displays an image on the web page.
<a> Anchor Creates a link to another page or to a location in the current page.
<abbr> Abbreviation Indicates an acronym or abbreviation of a longer word or phrase.
<acronym> Acronym Creates text that will be displayed when hovered over.
<audio> Audio Represents an interface for adding audio content to the page.
<br> In-line Break Represents a break in text. It is used when text needs to span multiple lines rather than being in-line.
<button> Button Represents a button meant to be clicked by the user.
<datalist> Data List Displays pre-defined values to a user while typing into an input box.
<dl> Descrition List Displays terms and details.
<dt> Description Term Describes a term found inside a <dl> tag. It usually comes with at least one corresponding <dd> details tag.
<dd> Description Describes details found inside a <dl> element. It usually comes with at least one corresponding <dt> term tag.
<del> Deleted Shows text that is to be removed from a document, usually rendered in strike-through text.
<div> Division Represents a generic division of content. It has no semantic meaning, but will separate its contents from the rest of the document.
<embed> Embed Inserts external content such as a video, image, page, and more.
<form> Form Represents an interface to collect and submit user supplied information. This can include open ended text inputs, radio buttons, calendar information, and more.
<hr> Horizontal Line Break Represents a semantic, horizontal line break between text elements.
<iframe> Embed Webpage Represents a container used to embed a second web page inside the current one. It can be used for content from the same domain as the parent, or even from a second domain.
<input/> Input Creates an interactive element, usually used within a form to allow user input. It can be used to make text boxes, color pickers, date pickers, and other UI elements.
<kbd> Keyboard Emphasizes characters to look like keys on a keyboard.
<label> Label Identifies captions for other elements in an HTML document.
<link> Link Connects the current page with an external file.
<meta> Metadata Represents an interface to provide metadata pertaining to the document. Metadata is data that is used to describe the document that contains it.
<object> Object Represents an external resource such as an image, a nested browsing context, or content to be handled by a browser plugin.
<option> Option Represents one option in a dropdown created by the select tag.
<output> Output Displays the result of a calculation or user action.
<param> Parameters Used to pass parameters to a resource defined in an object tag.
<picture> Pictures Source Represents multiple possible image sources to be applied to different devices and screen-sizes.
<s> Strike Through Represents strike-through text that is no longer need, accurate, or correct.
<select> Select Creates a drop-down list for the user to select from a number of option elements.
<span> Span Used for grouping related text or elements for styling and scripting.
<textarea> Text Area Displays multi-line plain-text input.
<track> Track An HTML element that specifies subtitles, closed captioning, and other text files for media elements.

Semantic HTML

Tag Name Description
<article> Article Represents a part of a page which is self-contained and could be published elsewhere. Common uses include blog posts or magazine articles.
<aside> Aside Represents a part of a page which is relevant to the current content, but not a part of it directly. It can be used for related links,
for clarifying a statement from the current article, or even for advertising meant for the current page.
<details> Details Encapsulates the <summary> element and any additional details which are only visible when the element is in an open state.
<figcaption> Figure Caption Describes the media encapsulated within the <figure> element.
<figure> Figure Encapsulates media such as an image, diagram, or code snippet.
<footer> Footer Represents a part of a page which is meant to be at the end of a completed block of content. Common uses include copyright information for the page or additional links to relevant pages.
<header> Header Represents a part of a page which is meant to be introductory. It can include heading tags, a logo, a search bar, and navigation elements.
<main> Main Represents the primary content within the body element of the web page.
<mark> Marked/Highlighted Represents part of a text that should be rendered as marked or highlighted.
<nav> Navigation Defines a block of navigation links such as menus and tables of contents.
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<summary> Summary Acts as a label for the <details> element and reveals additional information when clicked.
<time> Time Represents a time-related piece of information and can take a datetime attribute that makes it machine-readable.
<video> Video Embeds a video within the webpage.

Tables

Table tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
<th> Table head cell The header cell in a table.

Table attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
